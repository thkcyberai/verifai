# VerifAI

AI-powered fact-checking app - Instantly verify claims using LLM reasoning, web search, and media forensics.

## Quick Start
```bash
# Setup
cp .env.sample .env
make setup

# Start services
make services-up

# Run backend (Terminal 1)
make run-backend

# Run mobile (Terminal 2)
make run-mobile
```

See GETTING_STARTED.md for complete instructions.
